第二章 GitBook使用

第二章 GitBook使用

   本章节主要概括GitBook的基本使用，以及GitBook项目结构剖析。 GitBook基础用法非常简单基本上只有五步：

>

  1. 使用 gitbook init 初始化书籍目录

  2. 使用macdown等编辑工具对书籍内容布局编写

  3. 使用 gitbook build 生成静态网页

  4. 使用 gitbook serve 生成静态网页并可以在浏览器中运行

  5. 亦可使用GitHub与GitBook 将书籍部署到个人网站
